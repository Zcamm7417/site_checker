site_checker is a site connectivity checker utility . it takes one or more website Urls and check if those sites are online or offline.


it can perform the check synchronously and asynchronously.
# launching site_checker
launch site_checker by running sitechecker  on your terminal 
and passibg in the argument 
Arguments:
RP Checker provides the following options:- `-u` or `--urls` takes one or more URLs and checks if they're online.- `-f` or `--input-file` takes a file containing a list of URLs to check.- `-a` or `--asynchronous` runs the check asynchronously.
